# Code for assignment 3 for the Analyzing Data course

This repository contains the code that genre classification using `ministral-3:14b` [Assignment 3.ipynb](https://github.com/tijn-donners/analyzing_data_assignments/blob/master/assignment3/Assignment_3.ipynb)

`/data` contains the data used in `Assignment 3.ipynb`

`/results` contains both the dataset with genre classification for both few and zero shot prompting approaches as well as the evaluation reports, all in `.csv` fromat

### Setup

This code deploys remote computing  by using cloud models from [Ollama](ollama.com) (free tier). The setup instructions are included in the jupyter notebook. Within the notebook you can also configure a local model.
